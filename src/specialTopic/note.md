🔁 Autoboxing
Automatically converting a primitive type to its wrapper class.

Example:

int num = 5;
Integer obj = num;  // Autoboxing: int → Integer

-------------------------------------------------------------------------------------------------------------------------------------------------------------------

🔁 Unboxing
Automatically converting a wrapper class back to a primitive.

Example:

Integer obj = 10;
int num = obj;  // Unboxing: Integer → int

-------------------------------------------------------------------------------------------------------------------------------------------------------------------
